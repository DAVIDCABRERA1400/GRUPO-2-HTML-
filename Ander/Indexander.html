Introduccion a la Base de Datos

¿Que es una Base de Datos?

Creado el 01-05-2024

Las bases de datos se componen de varios elementos clave:

  • Datos: Son la información que se almacena en la base de datos. Pueden ser números, texto, imágenes, vídeos, etc.
  • Tablas: Las tablas son la forma en que se organiza la información en una base de datos relacional. Cada tabla consta de filas y columnas, donde cada fila representa un registro único y cada columna representa un atributo específico de esos registros.
  • Campos: Son las columnas de una tabla que representan los diferentes atributos de los registros. Por ejemplo, en una tabla de empleados, los campos podrían ser el nombre, el apellido, el salario, etc.
  • Registros: Son las filas individuales de una tabla que contienen los datos específicos de cada entidad. Por ejemplo, cada empleado en una tabla de empleados sería un registro.
  • Claves primarias y claves foráneas: La clave primaria es un campo o una combinación de campos que identifica de manera única cada registro en una tabla. Las claves foráneas son campos en una tabla que hacen referencia a la clave primaria de otra tabla y se utilizan para establecer relaciones entre tablas.
  • Consultas: Las consultas son instrucciones que se utilizan para recuperar, actualizar o eliminar datos de una base de datos.
  • Sistemas de gestión de bases de datos (DBMS): Son programas que permiten crear, mantener y manipular bases de datos. Algunos ejemplos populares son MySQL, PostgreSQL, Oracle y SQL Server.
