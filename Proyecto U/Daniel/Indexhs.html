HARDWARE

Creado el 01-05-2024

Es la parte física de un ordenador o sistema informático. Está formado por los componentes de material en estado físico. El término hardware viene del inglés, significa partes duras. Es utilizado para aludir a los componentes de carácter material que conforman un equipo de computación.

SOFTWARE

Creado el 01-05-2024

Es un término informático que hace referencia a un programa o conjunto de programas de cómputo, así como datos, procedimientos y pautas que permiten realizar distintas tareas en un sistema informático. Comúnmente se utiliza este término para referirsea los programas de un dispositivo informático, sin embargo, el software abarca todo lo que no se puede tocar en un sistema.
